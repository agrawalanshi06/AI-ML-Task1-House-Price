# AI-ML-Task1-House-Price
House Price Prediction using Linear Regression
## Overview
This project predicts house prices using Linear Regression.

## Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn

## Model Performance
- R² Score: 0.60

## How to Run
1. Open notebook
2. Run all cells
3. Enter input values

## Model
Model is saved as model.pkl using pickle.

- House Age
- Average Rooms
- Bedrooms
- Population
- Occupancy
- Latitude
- Longitude
